Start a new project
-------------------

You can clone the repository (or extract it in your new project directory).

* To clone ::

    $ hg clone https://........ atuitter
	
* Or extract ::

    $ tar xzf flask-atuin.tgz && mv flask-atuin atuitter

*In Windows you must know how to do that commands.*
